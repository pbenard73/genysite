Get Started

Steps

  1. npm install -g genysite
  2. Create your site folder
  3. Create the src/pages/ folder, and set your nunjucks file inside it, subfolder, for arboresence
  4. If you wanna use a template, set it inside the src/template folder
  5. If you need some assets for your website, set it inside the src/assets folder
  6. If your template need some assets, set it inside the src/template/assets folder
  7. Run genysite inside your root folder

Configuration File

You can provide a genysite.json configuration file in order to modify the parameters of the build

The json file had the following entries

  • homepage The prefix of all url, links inside the pages.
  • dist The folder's name of build

Template Filter

  • link contact.html will generate the path to the contact page src/pages/contact.njk
  • assets(isTemplateAssets = false)
